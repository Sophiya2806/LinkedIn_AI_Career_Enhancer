# LinkedIn AI Career Enhancer

An AI-powered tool that analyzes LinkedIn profiles and provides personalized career improvement suggestions.

## 🚀 Features
- Profile analysis
- AI-based suggestions
- Resume improvement tips
- Career guidance

## 🛠 Tech Stack
- React (Vite)
- Tailwind CSS
- Node.js (if backend)
- Supabase (if used)
